# AI-Powered Parametric Insurance for Gig Workers

## 👤 Persona
Food Delivery Partners (Swiggy/Zomato)

## ❗ Problem Statement
Gig workers lose income due to external disruptions like heavy rain, heatwaves, floods, and curfews. They currently have no financial protection.

## 💡 Solution
We propose an AI-powered parametric insurance platform that automatically detects disruptions and provides instant payouts for income loss.

## ⚙️ How It Works
1. The system monitors weather and location data.
2. If disruption (e.g., heavy rain) crosses a threshold, it triggers automatically.
3. AI calculates income loss based on past earnings.
4. Instant payout is credited to the worker.

## 💰 Weekly Pricing Model
- Premium: ₹50–₹100 per week
- Based on worker activity and city risk levels

## ⚡ Parametric Triggers
- Rainfall above threshold
- Temperature above 45°C
- Flood alerts
- Government curfews

## 🤖 AI Usage
- Risk prediction using historical data
- Dynamic premium calculation
- Fraud detection (location/activity validation)

## 🛠️ Tech Stack
- Frontend: React
- Backend: Node.js
- Database: MongoDB
- APIs: Weather API

## 🔄 Workflow
User registers → Selects plan → System monitors → Trigger occurs → Claim auto-generated → Instant payout

## 📌 Conclusion
This system ensures financial safety for gig workers by providing quick and automatic compensation during unexpected disruptions.
